# DVD Spring Boot Project
Учебное веб-приложение на **Spring Boot** для учёта DVD-дисков.

В проекте реализованы:

- регистрация и авторизация пользователей;
- роли пользователей: `USER` и `ADMIN`;
- личный кабинет пользователя;
- просмотр свободных DVD-дисков;
- взятие свободного диска текущим пользователем;
- возврат диска владельцем;
- CRUD для DVD-дисков для администратора;
- назначение новых администраторов;
- обработка ошибок через @ControllerAdvice и @ExceptionHandler;
- хранение данных в PostgreSQL.

## Стек технологий

- Java 17
- Spring Boot
- Spring Web MVC
- Spring Security
- Spring Data JPA
- Hibernate
- PostgreSQL
- Thymeleaf
- Bean Validation
- ModelMapper
- Maven

## Функциональность

### Для неавторизованного пользователя

Неавторизованный пользователь может:

- открыть страницу входа;
- зарегистрироваться;
- просматривать страницы ошибок.

### Для пользователя с ролью `USER`

Пользователь может:

- открыть личный кабинет;
- просматривать список свободных дисков;
- просматривать страницу конкретного диска;
- взять свободный диск;
- вернуть свой диск;
- выйти из аккаунта.

### Для администратора с ролью `ADMIN`

Администратор может всё, что может обычный пользователь, а также:

- создавать DVD-диски;
- редактировать DVD-диски;
- удалять DVD-диски;
- назначать других пользователей администраторами.
